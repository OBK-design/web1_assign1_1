The Dl, DT & DD tags:
Definition of the definition list, term, and description tags.
The BR tag:
The Line break tag helps in creating line breaks within the text.
The Nav tag:
The Navigation tag is used to set up navigation links within the webpages.
The OL, UL & LI tags:
They stand for, Ordered List (numbered) and Unordered List (bulleted) tags respectively,
listed Item define individual items inside those lists.
The Footer tag:
The Footer tag is used to define the footer of a document or section, see it as the Legs of the webpage.
